<h2 align="center">Como Usar 🤔</h2>

   ```
   - Clone esse repositório:
   $ git clone https://github.com/alexandref13/gameplay
   - Entre no diretório:
   $ cd gameplay
   - Instale as dependências:
   $ flutter pub get
   - Inicie o app: 
   $ flutter run
   ```
